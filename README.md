# Task Manager 프로젝트

### 설명
팀 단위 간단한 진행 상황 표시를 편리하게 할 수 있도록 해주는 서비스이다.
주 목적은 온라인 강의 환경에서 실습 진행 상황을 편리하게 파악할 수 있도록 하는 것이다.
